**Interactive-Sales-Analysis-Dashboard**
 I built this project to practice data analysis, data cleaning, and dashboard design skills using Excel. It helps in identifying sales trends, top-performing products, and regional performance in a simple and user-friendly way.  The dashboard is fully interactive and updates automatically based on user selection.
Interactive Sales Analysis Dashboard

This project is an Excel-based interactive dashboard used to analyze sales performance.

**Features**
- KPI cards (Sales, Profit, Quantity)
- Interactive slicers
- Dynamic charts
- Clean dashboard design

**Tools Used**
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
